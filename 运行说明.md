# 中国象棋 Web UI 运行说明

## 一键启动方法（推荐）

### 方法一：双击启动（最简单）

1. 进入项目目录：`e:\program\ysj-0006`
2. 双击 `启动.bat` 文件
3. 等待自动完成环境检查和依赖安装
4. 看到提示后，在浏览器中打开 http://localhost:5000

---

### 方法二：CMD 命令行启动

打开 CMD（命令提示符），执行以下命令：

```cmd
cd /d e:\program\ysj-0006
启动.bat
```

或者手动执行：

```cmd
cd /d e:\program\ysj-0006
pip install -r requirements.txt
python app.py
```

## 功能介绍

### 核心功能

1. **人机对战**: 点击「人机对战」按钮，选择执红或执黑，开始与AI对弈
2. **预测未来五步**: 点击「预测未来五步」，AI将分析并展示未来5步的最佳走法
3. **分析当前局面**: 点击「分析当前局面」，查看前10个最佳候选走法及评分
4. **查看棋盘**: 主界面中央实时显示当前棋盘状态
5. **输入走法**: 在左侧面板输入坐标（起行 起列 目标行 目标列），点击「执行走法」
6. **悔棋**: 点击「悔棋」按钮，撤销上一步走法（人机模式撤销两步）
7. **重新开始**: 点击「重新开始」，重置棋盘到初始状态
8. **退出**: 点击「退出」，退出游戏
9. **规则展示**: 点击「规则展示」，查看各棋子的走法规则

### 棋盘操作

- **点击棋子**: 点击己方棋子，会显示该棋子可以走到的位置
  - 绿色圆点：可以移动到的空位
  - 红色圆圈：可以吃掉的对方棋子
- **点击目标位置**: 点击显示的可走位置，完成走子
- **点击空白处**: 取消选中的棋子
- **最后走法标记**: 金色方框标记上一步的起止位置

### 界面信息

- **左侧面板**:
  - 对局信息：当前回合、走子步数、局面评估
  - 游戏控制：所有功能按钮
  - 输入走法：手动输入坐标走子
- **右侧面板**:
  - 消息提示：显示操作记录和系统消息
  - 已吃棋子：显示双方被吃掉的棋子

## 常见问题

### Q: 双击启动.bat没反应？
A: 右键点击「启动.bat」，选择「以管理员身份运行」

### Q: 提示找不到Python？
A: 请先安装Python 3.6或更高版本，下载地址：https://www.python.org/downloads/
   安装时请勾选「Add Python to PATH」

### Q: 依赖安装失败？
A: 检查网络连接，或使用国内镜像：
   ```cmd
   pip install -i https://pypi.tuna.tsinghua.edu.cn/simple flask flask-cors
   ```

### Q: 浏览器打不开页面？
A: 确认服务器已启动，尝试刷新页面或使用 http://127.0.0.1:5000

### Q: AI思考太慢？
A: AI搜索深度默认为4步，如需更快可以修改 `app.py` 中的深度设置：
   ```python
   ai = ChessAI(depth=3)  # 修改为3，速度更快
   ```
